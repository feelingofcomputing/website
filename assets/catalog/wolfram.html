Wolfram Language

Reviewed July 17, 2019

Wolfram Language, a confusing rebranding of Mathematica, is a programming language and notebook-based browser environment. It has 5,000 functions and data sources built into the language -- no imports required. It has broad applications in engineering, sciences, mathematics, finance and education.

Contextual Documentation

Hovering over any Wolfram function brings up contextual documentation, which you can click on for further reading.

Embedded OAuth

Evaluating certain expressions, such as SocialMedia["Twitter"], automatically start OAuth flows to login to the third-party application from Wolfram to be able to pull data from that app. This makes it incredibly easy to use basic integrations. Unfortunately, they were limited in their actual usefulness, being able to only pull limited subsets of user data.
